# Pygame of Life

Conway's Game of Life simulation built in Pygame

![ezgif com-video-to-gif-converter](https://github.com/user-attachments/assets/9b5a758e-f6b8-4ded-8f27-a05ab859678d)

| Controls   | Description |
| --------- | ------- |
| Mouse Click     | Add Cell|
| SPACEBAR | (Un)Pause|  
| UP | Speed UP   |
| DOWN | Slow DOWN|
| R | Randomize Board|
| C | Clear Board|
